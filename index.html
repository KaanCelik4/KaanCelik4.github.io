İçerik

Bu site bir bilgisayar mühendisi öğrencisinin Web Programlama dersinde yaptığı uygulamaları temsil eder.

Ders kapsamında HTML5 yapısı, CSS3 stilleri ve modern web standartları üzerine çalışmalar gerçekleştirilmektedir.

Bilgi

Web programlama, internet üzerinde çalışan uygulamaların geliştirilmesi sürecidir. Bu süreç hem görsel tasarımı (Frontend) hem de sunucu tarafındaki işlemleri (Backend) kapsar.

Günümüzde modern web geliştirme; HTML, CSS ve JavaScript temelleri üzerine inşa edilen dinamik yapılarla şekillenmektedir.
